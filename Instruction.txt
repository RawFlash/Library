Инструкция для пользователя системы.

Система является автоматизированным рабочим местом сотрудника библиотеки.

Система разработана для сотрудников библиотеки.

Система предполагает следующие варианты использования:
1) Отметка о выдаче книги для использования "на руки" читателю.
2) Отметка о выдаче книги для использования в зале библиотеки.
3) Отметка о возвращении книги в библиотеку.

1) Отметка о выдаче книги для использования "на руки" читателю.

Данный сценарий предполагает: 
1. Переход в раздел "Выдать книгу на руки".
2. Выбор книги, которая выдана.
3. Выбор читателя, которому книга выдана.
4. Действие завершено.

2) Отметка о выдаче книги для использования в зале библиотеки.

Данный сценарий предполагает: 
1. Переход в раздел "Выдать книгу в библиотеке".
2. Выбор книги, которая выдана.
3. Выбор читателя, которому книга выдана.
4. Действие завершено.

3) Отметка о возвращении книги в библиотеку.
1. Переход в раздел "Вернуть книгу".
2. Выбор книги, которая будет возвращена.
3. Действие завершено.

Обратите внимание: 
1) Возврат книг возможен только для тех книг,
которые были выданы "на руки" или в библиотеке.
2) Выдача книг возможна только в том случае,
если книга находиться в библиотеке.
